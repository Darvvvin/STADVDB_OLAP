# Setup

**INIT Postgres by looking at the migations/init.sql**

##### Server
```
/server$ yarn
/server$ yarn migrate-up

/client$ yarn
===OR===
/server$ npm i
/server$ npm run migrate-up

/client$ npm i
```

# DEV

**SERVER**
```
/server$ yarn dev
===OR===
/server$ npm run dev
```

**CLIENT**
```
/client$ yarn dev
===OR===
/client$ npm run dev
```
